    # puppeteer

    Puppeteer 是一个 Node 库，它提供了高级 API 来通过 DevTools 协议控制 Chrome 或 Chromium。

    生成页面的屏幕截图和 PDF； 爬取 SPA 并生成预渲染的内容（即 SSR）； 自动进行表单提交，UI 测试，键盘输入等； 创建最新的自动化测试环境。使用最新的 JavaScript 和浏览器功能，直接在最新版本的 Chrome 中运行测试； 捕获时间线跟踪网站，以帮助诊断性能问题； 测试 Chrome 扩展程序。

    从上可见，Puppeteer 可以实现在Node 端生成页面的 PDF 功能。
